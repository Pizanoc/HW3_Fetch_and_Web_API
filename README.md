# HW3_Fetch_and_Web_API
For this assignment, I created a web page that uses the Rick and Morty API to search for characters by name. The user enters a character name into a form, and JavaScript validates the input to make sure it is not empty. When the form is submitted, the page uses fetch() to retrieve character data from the API and display the results in a user-friendly card format. The page shows each character’s image, name, status, species, gender, origin, and current location. I used HTML for the structure, CSS for styling, and JavaScript for validation, event handling, and API retrieval.
